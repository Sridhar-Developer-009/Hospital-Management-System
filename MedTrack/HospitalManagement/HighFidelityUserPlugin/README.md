# HighFidelityUserPlugin

Figma plugin that creates a single editable 1024 x 1536 MedTrack landing page frame.

## Import in Figma

1. Open Figma Desktop.
2. Go to Plugins -> Development -> Import plugin from manifest.
3. Select `manifest.json` from this folder.
4. Run `HighFidelityUserPlugin`.

The plugin creates one frame named `MedTrack Landing Page - Fully Editable`.

Most interface elements are editable Figma layers. Photo-heavy sections are separate image layers so the high-fidelity visual quality is preserved.
